## 1. 프로젝트 개요

- **구분**: 팀 프로젝트 (프론트엔드 담당 / 기획 참여)
- **기간**: 2025.05
- **기여도**: 프론트엔드 개발 기여도 100% (팀 구성원: 프론트엔드 1명, 백엔드 2명, 디자이너 1명)
- **기술 스택**: React, TypeScript, Tailwind CSS, Web Speech API (TTS/STT), Google Gemini API
- **주요 성과**:
  - Google APAC Solution Challenge 참가 및 성공적인 기술 데모 수행

---

## 2. 주요 역할 및 기술적 기여

### 🗣️ 노약자 친화적 대화형 UX 및 실시간 음성 상호작용
- **도전 과제**: 정보통신 기기에 익숙하지 않은 노약자들이 타이핑이나 복잡한 설정 없이도 서비스에 손쉽게 적응할 수 있도록 설계해야 했습니다.
- **해결 전략**:
  - `Web Speech API`를 연동하여 마이크 터치 한번으로 음성 입력을 텍스트로 전환(STT)하고, 생성된 AI 답변을 편안한 한국어 음성(TTS)으로 출력하는 완전 음성 대화 UX를 구현했습니다.
  - 마이크 활성화 여부와 음성 인식 중 상태를 직관적으로 파악할 수 있는 스피커 웨이브 모션 등 풍부한 마이크로 애니메이션을 가미해 시각적 피드백을 극대화했습니다.

### 🧩 화면 구조 설계 및 유연한 프론트엔드 통합
- **도전 과제**: 기획 및 디자이너 팀원과의 요구사항을 신속하게 조율하면서 개발 기간에 맞춰 통합을 완료해야 했습니다.
- **해결 전략**:
  - 1인 프론트엔드 개발자로서 서비스 기획부터 세부 화면 레이아웃 정의, API 정합성 검증까지 리드했습니다.
  - 모바일 웹 환경에서도 부드럽게 렌더링되도록 스파게티성 CSS를 배제하고 Tailwind CSS 기반의 확장 가능한 구조로 뼈대를 다졌습니다.
